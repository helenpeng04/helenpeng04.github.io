This project investigates how incentives influence visual search performance. 
The task was designed using **Gorilla**, and data analysis was conducted in **Jamovi**. 
The analysis explores the relationship between rewards and search efficiency.

**Collaborators**:\
*Cami Streuly, Shirley Du, Cole Kaforey, Abigail Noyce*
